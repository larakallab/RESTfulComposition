* Dependencies:
- flask
- flask_restful
- snakestoolkit ( http://code.google.com/p/python-snakes/ ) : to build petri nets model
- neco (https://github.com/Lvyn/neco-net-compiler) : to compile petri net + follow the instructions presented in: https://code.google.com/archive/p/neco-net-compiler/wikis/UsingNecoCLI.wiki

* Optional:
- GraphViz (http://graphviz.org/) : to be able to draw nets
